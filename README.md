# 🤖 Agentic AI From Scratch

> Learn how modern AI Agents work by building every component from scratch using Python.

Instead of relying on frameworks like LangChain or CrewAI from the beginning, this repository focuses on understanding the **core building blocks** behind modern AI agents.

You will progressively build an AI Agent by implementing each component yourself, gaining a deep understanding of how reasoning, tool usage, context management, and function calling work internally.

# 📚 Module Overview

## 📖 Module 1 — Introduction

Learn the fundamentals of Agentic AI, how AI Agents differ from traditional chatbots, and the architecture of modern autonomous systems.

---

## 📡 Module 2 — LLM API

Learn how applications communicate with Large Language Models using APIs.

Topics include:

- OpenRouter
- API Requests
- Responses
- Temperature
- Tokens

---

## 🛠 Module 3 — Tool Calling

Teach an LLM to interact with external tools.

Topics include:

- JSON Schema
- Function Calling
- Tool Execution
- Tool Results

---

## 🤖 Module 4 — ReAct Agent

Build an AI Agent capable of reasoning before acting.

Topics include:

- Thought
- Action
- Observation
- Agent Loop

---

## 📊 Module 5 — Agent Tracing

Understand what happens internally while an AI Agent executes.

Topics include:

- Trace Logs
- Token Usage
- Context Engineering
- Cost Analysis

---

## 💻 Module 6 — Code Agent

Give your AI Agent the ability to:

- Read Files
- Write Files
- Execute Python
- Explore Directories

---

## ⚡ Module 7 — Native Function Calling

Learn how modern LLMs perform structured function calling without manually parsing responses.

# 🎓 Learning Philosophy

This repository follows a **learn by building** approach.

Rather than hiding complexity behind frameworks, every module explains the underlying concepts and implements them step by step.

Once these foundations are understood, transitioning to frameworks such as LangGraph, CrewAI, or the OpenAI Agents SDK becomes significantly easier.
